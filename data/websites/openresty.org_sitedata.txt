href=\"luajit.html\">LuaJIT</a>, many carefully written Lua libraries, lots of high quality <a href=\"components.html\">3rd-party Nginx modules</a>, and most of their external dependencies. It is designed to help developers easily build scalable web applications, web services, and dynamic web gateways.</p>\n<p>By taking advantage of various well-designed <a href=\"nginx.html\">Nginx</a> modules (most of which are developed by the OpenResty team themselves), OpenResty<sup>®</sup> effectively turns the nginx server into a powerful web app server, in which the web developers can use the Lua programming language to script various existing nginx C modules and Lua modules and construct extremely high-performance web applications that are capable to handle 10K ~ 1000K+ connections in a single box.</p>\n<p>OpenResty<sup>®</sup> aims to run your server-side web app completely in the <a href=\"nginx.html\">Nginx</a> server, leveraging <a href=\"nginx.html\">Nginx</a>'s event model to do non-blocking I/O not only with the HTTP clients, but also with remote backends like MySQL, PostgreSQL, Memcached, and Redis.</p>\n<p>Real-world applications of OpenResty<sup>®</sup> range from dynamic web portals and web gateways, web application firewalls, web service platforms for mobile apps/advertising/distributed storage/data analytics, to full-fledged dynamic web applications and web sites. The hardware used to run OpenResty<sup>®</sup> also ranges from very big metals to embedded devices with very limited resources. It is not uncommon for our production users to serve billions of requests daily for millions of active users with just a handful of machines.</p>\n<p>OpenResty<sup>®</sup> is <em>not</em> an <a href=\"nginx.html\">Nginx</a> fork. It is just a software bundle. Most of the patches applied to the <a href=\"nginx.html\">Nginx</a> core in OpenResty<sup>®</sup> have already been submitted to the official <a href=\"nginx.html\">Nginx</a> team and most of the patches submitted have also been accepted. We are trying hard <em>not</em> to fork <a href=\"nginx.html\">Nginx</a> and always to use the latest best <a href=\"nginx.html\">Nginx</a> core from the official <a href=\"nginx.html\">Nginx</a> team.</p>\n<p>See <a href=\"components.html\">Components</a> for the complete list of software bundled in OpenResty<sup>®</sup>.</p>\n<p>See <a href=\"getting-started.html\">GettingStarted</a> on how to quickly setup an OpenResty<sup>®</sup> server that can say hello world over HTTP.
