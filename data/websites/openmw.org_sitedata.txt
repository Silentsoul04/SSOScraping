this? I can see the Ghostfence all the way from Ald-Ruhn and my fps is above 10. This is OpenMW and not MGE XE, right?</p><p>This is OpenMW alright and <a href=\"http://i.imgur.com/IuRDXv7.jpg\">what you’re seeing</a> is made possible by the newest addition from scrawl – distant terrain. This new feature is available in the latest OpenMW commit and if you’re on windows, you can download it from <a href=\"https://forum.openmw.org/viewtopic.php?f=20&amp;t=1808#p18839\">Ace’s nightly builds</a> to try it out yourself.</p><p>So what does it do? In short, it <a href=\"http://i.imgur.com/YklBk3E.jpg\">simplifies the land mesh</a>, nothing else. It doesn’t touch on scripts, NPCs and statics, but having simplified land certainly lightens the workload to a point where we can have some wiggle room with our view distance.</p><p>You may have noticed that I’ve taken use of this in some of the screenshots and loaded enough statics to create the illusion that there are objects <a href=\"http://i.imgur.com/HLZzgDp.jpg\">as far as the eye can see</a>. You can only spot <a href=\"http://i.imgur.com/pAg1oKv.jpg\">empty land</a> if I purposefully look at something very distant.</p><p>I must admit, I cheated a little to boost my fps. There is an additional feature called small object culling, which makes it so that objects too tiny are not rendered and my computer’s performance is spared. I used this rather aggressively, as I am working with a few years old laptop, but someone with a dedicated gaming rig might get better results while culling less.</p><p>Finally, I would like to share my <a href=\"http://i.imgur.com/iFuVAdf.jpg\">last screenshot</a>, as well as how you can set this up. After downloading and installing the latest commit, find settings.cfg and add the following lines:</p><p>[Camera]</p><p>viewing distance = 6666.0 #Determines how much land is rendered. Add another 6 or 66 to it.</p><p>small feature culling pixel size = 2.0 #The default is 2, but you may want to change it to 16 at the highest. Improves performance.</p><p>[Cells]</p><p>exterior cell load distance = 1 #This loads statics, NPCs and scripts around you. You may want to set this to 2, 3 or 4.</p><p>[Terrain]</p><p>distant terrain = True #The on-switch to distant terrain.
