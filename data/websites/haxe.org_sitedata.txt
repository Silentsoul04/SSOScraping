Language\n\t\t\t\t</div>\n\n\t\t\t\t<div class=\"span6 desc\">\n\t\t\t\t\tThe Haxe programming language is a high level strictly typed programming language which is used by the Haxe compiler to produce cross-platform native code.\n\t\t\t\t\tThe Haxe programming language is easy to learn if you are familiar already with either Java, C++, PHP, AS3 or similar object oriented languages.\n\t\t\t\t\tThe Haxe programming language has been especially designed in order to adapt the various platforms native behaviors and allow efficient cross-platform development.\n\t\t\t\t</div>\n\n\t\t\t\t<div class=\"span4 links\">\n\t\t\t\t\t<ol>\n\t\t\t\t\t\t<li><a href=\"/documentation/introduction/toolkit-introduction.html\">Introduction to the Haxe or binary. Each platform is natively supported, without any overhead coming from running inside a virtual machine. The Haxe Compiler is very efficient and can compile thousands of classes in seconds.\n\t\t\t\t</div>\n\n\t\t\t\t<div class=\"span4 links\">\n\t\t\t\t\t<ol>\n\t\t\t\t\t\t<li><a href=\"/manual/compiler-usage.html\">Using the Haxe Compiler</a></li>\n\t\t\t\t\t\t<li><a href=\"/documentation/introduction/compiler-targets.html\">List of Supported Platforms</a></li>\n\t\t\t\t\t\t<li><a href=\"/manual/cr-features.html\">The Haxe Compiler features</a></li>\n\t\t\t\t\t</ol>\n\t\t\t\t</div>\n\t\t\t</div>\n\n\t\t\t<div class=\"row component\">\n\t\t\t\t<div class=\"span2 title\">\n\t\t\t\t\tThe Haxe standard library\n\t\t\t\t</div>\n\n\t\t\t\t<div class=\"span6 desc\">\n\t\t\t\t\tThe Haxe standard library provides a common set of highly tested APIs that gives you complete cross-platform behavior.
