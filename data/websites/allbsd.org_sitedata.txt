<p><tt>anoncvs.allbsd.org</tt> provides read-only anonymous CVS access of the following\n                    repositories: FreeBSD, NetBSD, OpenBSD, and DragonFly BSD. The server uses SSH2 password\n                    authentication on port 2022 (not pserver and rsh).</p>\n\n                    <h4>How To Use</h4>\n\n                    <p><tt>anoncvs.allbsd.org</tt> supports OpenSSH version 3.5 or later only. For the other OpenSSH\n                    versions and/or other SSH implementations, this may or may not work. The following instruction\n                    assumes you use OpenSSH.</p>\n\n                    <p>Put the following lines into your <tt>${HOME}/.ssh/config</tt>.</p>\n<pre class=\"sidebar\">Host                    anoncvs.allbsd.org\nPort                    2022\nProtocol                2\n</pre>\n\n                    <p>After that, you can access to <tt>anoncvs.allbsd.org</tt>. Set ${CVSROOT} and ${CVS_RSH} like\n                    this:</p>\n<pre class=\"sidebar\"># For all\nCVS_RSH=ssh\n\n# For FreeBSD\nCVSROOT=anoncvs-freebsd@anoncvs.allbsd.org:/home/ncvs\n\n# For NetBSD\nCVSROOT=anoncvs-netbsd@anoncvs.allbsd.org:/cvsroot\n\n# For OpenBSD\nCVSROOT=anoncvs-openbsd@anoncvs.allbsd.org:/cvs\n\n# For DragonFly BSD\nCVSROOT=anoncvs-dfly@anoncvs.allbsd.org:/dcvs\n</pre>\n\n                    <p>The access uses OpenSSH password authentication. However, since the password on\n                    <tt>anoncvs.allbsd.org</tt> is empty string, you do not need to enter the password. If you are in\n                    trouble, please contact admin@allbsd.org.</p>\n\n                    <h3><a id=\"PUB-RSYNC\" name=\"pub-rsync\">Public rsync service</a></h3>\n\n                    <p><a href=\"rsync://rsync.allbsd.org/\">rsync.allbsd.org</a> provides the distribution and source\n                    trees of FreeBSD, NetBSD, OpenBSD, DragonFly BSD, OpenDarwin, XFree86, TenDRA, CVSync, and so on.\n                    Available collections and their upstreams are found by using \"finger @info.allbsd.org\" or \"rsync\n                    rsync://rsync.allbsd.org/\".</p>\n\n                    <h3><a id=\"PUB-CVSYNC\" name=\"pub-cvsync\">Public cvsync service</a></h3>\n\n                    <p><a href=\"http://www.cvsync.org/\">cvsync</a> (<a href=\"http://pub.allbsd.org/cvsync/\">local\n                    mirror</a>) is another CVS repository synchronization utility by MAEKAWA Masahide (maekawa AT\n                    cvsync DOT org), which is very similar to CVSup but not compatible with it, and written in C and\n                    pthreads (POSIX 1003.1c-1995, aka ISO/IEC 9945-1:1996) library. Although it is still at\n                    experimental stage, it has reasonable portability, better performance, and IPv6 capability.</p>\n\n                    <p><a href=\"cvsync/\">Other public cvsync mirrors are listed here.</a></p>\n\n                    <p><tt>cvsync.allbsd.org</tt> provides the distribution and source trees of FreeBSD, NetBSD,\n                    OpenBSD, DragonFly BSD, OpenDarwin, XFree86, TenDRA, and CVSync itself via cvsync. Available\n                    collections can be retrieved using <tt>cvsync cvsync://cvsync.allbsd.org/</tt> command.
