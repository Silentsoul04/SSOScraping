OpenVZ creates multiple secure, isolated Linux containers (otherwise known as VEs or VPSs) on a single physical server enabling better server utilization and ensuring that applications do not conflict. Each container performs and executes exactly like a stand-alone server; a container can be rebooted independently and have root access, users, IP addresses, memory, processes, files, applications, system libraries and configuration files. \n</p><p>OpenVZ is free open source software, available under GNU GPL.\n</p><p>OpenVZ is the basis of <a href=\"/Virtuozzo\" title=\"Virtuozzo\">Virtuozzo</a>, a virtualization solution offered by <a rel=\"nofollow\" class=\"external text\" href=\"http://www.virtuozzo.com/\">Virtuozzo</a> company. Virtuozzo is optimized for hosters and offers hypervisor (VMs in addition to containers), <a href=\"/Virtuozzo_Storage\" title=\"Virtuozzo Storage\">distributed cloud storage</a>, dedicated support, management tools, and easy installation.\n</p>\n<h2><span class=\"mw-headline\" id=\"Installation_and_usage\">Installation and usage</span></h2>\n<p>OpenVZ is easy to install if you already have Linux installed on your machine. See <a href=\"/Quick_installation\" title=\"Quick installation\">quick installation</a> for installation info. Alternatively, you can try a <a href=\"/Download_live_CD\" title=\"Download live CD\" class=\"mw-redirect\">live CD</a> to test drive the technology without a need to install anything.\n</p><p>A comprehensive (although somewhat old) <span class=\"PDFlink\"><a href=\"http://download.openvz.org/doc/OpenVZ-Users-Guide.pdf\" class=\"extiw\" title=\"download:doc/OpenVZ-Users-Guide.pdf\">doc/OpenVZ-Users-Guide.pdf</a></span><span style=\"font-size:smaller;\">&nbsp;( 1.3&nbsp;MB)</span> is a good start. This wiki also has a lot of information (use search function, see <a href=\"/Special:Categories\" title=\"Special:Categories\">list of all categories</a> or <a href=\"/Special:AllPages\" title=\"Special:AllPages\">list of all pages</a>.\n</p>\n<h2><span class=\"mw-headline\" id=\"Support\">Support</span></h2>\n<p>The supported version of OpenVZ is <a rel=\"nofollow\" class=\"external text\" href=\"https://virtuozzo.com/openvz\">Virtuozzo</a>.\n</p>\n<h3><span class=\"mw-headline\" id=\"Free_support\">Free support</span></h3>\n<p>Please read the documentation before asking questions.
