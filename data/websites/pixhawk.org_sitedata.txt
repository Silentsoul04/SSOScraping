alt=\"\"></a>\n</p>\n\n<p>\nPixhawk is an independent, open-hardware project aiming at providing high-end autopilot hardware to the academic, hobby and industrial communities at low costs and high availability. It provides hardware for the Linux Foundation <a href=\"http://dronecode.org\" class=\"urlextern\" target=\"_blank\" title=\"http://dronecode.org\" rel=\"nofollow\">DroneCode</a> project. It originated from the <a href=\"http://pixhawk.ethz.ch\" class=\"urlextern\" target=\"_blank\" title=\"http://pixhawk.ethz.ch\" rel=\"nofollow\">PIXHAWK Project</a> of the <a href=\"http://cvg.ethz.ch\" class=\"urlextern\" target=\"_blank\" title=\"http://cvg.ethz.ch\" rel=\"nofollow\">Computer Vision and Geometry Lab of ETH Zurich (Swiss Federal Institute of Technology)</a> and  <a href=\"http://www.asl.ethz.ch\" class=\"urlextern\" target=\"_blank\" title=\"http://www.asl.ethz.ch\" rel=\"nofollow\">Autonomous Systems Lab</a> as well from a number of excellent invididuals (<a href=\"/credits\" class=\"wikilink1\" title=\"credits\">Contact and Credits</a>).\n</p>\n\n<p>\n</p><p></p><div class=\"notetip\">\n<strong>We are always looking for new developers!</strong> Get in touch via the <img src=\"/_media/groups_button.png\" class=\"media\" alt=\"\"><a href=\"http://groups.google.com/group/px4users\" class=\"urlextern\" target=\"_blank\" title=\"http://groups.google.com/group/px4users\" rel=\"nofollow\">Forum (Web View)</a>.\n</div><p></p>\n<p></p>\n\n</div>\n\n<h2 class=\"sectionedit2\" id=\"autopilot_software\">Autopilot Software</h2>\n<div class=\"level2\">\n\n<p>\nThe Pixhawk autopilot module runs a very efficient real-time operating system (RTOS), which provides a POSIX-style environment (i.e. printf(), pthreads, /dev/ttyS1, open(), write(), poll(), ioctl(), etc). The software can be updated with an USB bootloader. We support multiple flight stacks: <a href=\"http://px4.io\" class=\"urlextern\" target=\"_blank\" title=\"http://px4.io\" rel=\"nofollow\">PX4</a> and <a href=\"http://ardupilot.org\" class=\"urlextern\" target=\"_blank\" title=\"http://ardupilot.org\" rel=\"nofollow\">APM</a>.\n</p>\n\n</div>\n\n<h2 class=\"sectionedit3\" id=\"autopilot_hardware\">Autopilot Hardware</h2>\n<div class=\"level2\">\n\n<p>\n<a href=\"/modules/pixhawk\" class=\"wikilink1\" title=\"modules:pixhawk\">PIXHAWK</a> is the industry standard autopilot, designed by the open hardware development team in collaboration with <a href=\"https://3dr.com\" class=\"urlextern\" target=\"_blank\" title=\"https://3dr.com\" rel=\"nofollow\">3D Robotics</a> and <a href=\"http://www.ardupilot.org\" class=\"urlextern\" target=\"_blank\" title=\"http://www.ardupilot.org\" rel=\"nofollow\">ArduPilot Group</a>.\n</p>\n\n<p>\n<a href=\"/modules/pixhawk\" class=\"media\" title=\"modules:pixhawk\"><img src=\"/_media/modules/pixhawk-logo-view.jpg?w=400&amp;tok=331821\" class=\"medialeft\" alt=\"\" width=\"400\"></a>\n<br>\n\n</p>\n<ul>\n<li class=\"level1\"><div class=\"li\"> 168 <abbr title=\"Megahertz\">MHz</abbr> Cortex M4F CPU (256 KB RAM, 2 <abbr title=\"Megabyte\">MB</abbr> Flash)</div>\n</li>\n<li class=\"level1\"><div class=\"li\"> Sensors: 3D ACC / Gyro / MAG / Baro</div>\n</li>\n<li class=\"level1\"><div class=\"li\"> Integrated backup, override and failsafe processor with mixing</div>\n</li>\n<li class=\"level1\"><div class=\"li\"> microSD slot, 5 UARTs, CAN, I2C, SPI, ADC, etc</div>\n</li>\n<li class=\"level1\"><div class=\"li\"> Order from:<br>\n</div>\n</li>\n</ul>\n\n<p>\n<a
