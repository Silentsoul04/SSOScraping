The most famous <a href=\"./Desktops/\">X desktops</a> are <a href=\"http://www.gnome.org\">GNOME</a> and <a href=\"http://www.kde.org\">KDE</a>, but developers working on any Linux/UNIX GUI technology are welcome to participate. </p>\n\n<p>freedesktop.org is building a base platform for desktop software on Linux and UNIX.  The elements of this platform have become the backend for higher-level application-visible APIs such as Qt, GTK+, XUL, VCL, WINE, GNOME, and KDE.  The base platform is both <a href=\"./Software/\">software</a> and <a href=\"./Specifications/\">specifications</a>. </p>\n\n<h2 id=\"software\">Software</h2>\n\n<p>freedesktop.org hosts any \"on-topic\" software projects.  If you have a project that fits into our mission and needs hosting, please make a request using our <a href=\"http://bugs.freedesktop.org\">bugzilla</a>. Mailing lists about freedesktop software are hosted <a href=\"http://lists.freedesktop.org/mailman/listinfo\">here</a>. </p>\n\n<h2 id=\"standards\">Standards</h2>\n\n<p>freedesktop.org is not a formal standards organization, though some see a need for one that covers some of the areas we are working on.  For Linux operating system standards, look at the <a href=\"http://www.linuxbase.org\">Linux Standard Base</a> project.  <a href=\"http://www.x.org\">The X.Org Foundation</a> and the <a href=\"http://www.ietf.org\">IETF</a> are other groups that do formal standards. The <a href=\"http://www.freestandards.org\">Free Standards Group</a> is one group that publishes \"de jure\" standards for free software; freedesktop.org is loosely affiliated with the FSG. </p>\n\n<p>Unlike a standards organization, freedesktop.org is a \"collaboration zone\" where ideas and code are tossed around, and de facto specifications are encouraged.  The primary entry to these discussions is the <a href=\"http://lists.freedesktop.org/mailman/listinfo/xdg\">xdg mailing list</a>. </p>\n\n<h2 id=\"gettinginvolved\">Getting Involved</h2>\n\n<p>You can get involved in a number of ways.  See the <a href=\"./MissionStatement/\">MissionStatement</a> for our principle activities.  See <a href=\"./GettingInvolved/\">GettingInvolved</a> for concrete suggestions on how you can contribute. See <a href=\"./AccountRequests/\">AccountRequests</a> for information on how to obtain commit access to a project.  See <a href=\"./NewProject/\">NewProject</a> for how to start a new project. </p>\n\n<h2 id=\"contributorcovenant\">Contributor Covenant</h2>\n\n<p>freedesktop.org has adopted the <a href=\"./CodeOfConduct/\">Contributor Covenant</a> for the fora it provides.
