It is the basis for multiple\ncommercial offerings.  It supplies the AAA needs of many Fortune-500\ncompanies and Tier 1 ISPs. </p>\n<p>It is also widely used for <a href=\"enterprise-wifi.html\">Enterprise\nWi-Fi and IEEE 802.1X</a> network security, particularly in the academic\ncommunity, including <a href=\"http://www.eduroam.org/\">eduroam</a>.</p>\n<p>The server is <a href=\"/features/fast.html\">fast</a>, <a href=\"/features/\">feature-rich</a>, <a href=\"/features/modules.html\">modular</a>, and <a href=\"/features/scalability.html\">scalable</a>.</p>\n\n<p>The server has reached a stable <a href=\"ftp://ftp.freeradius.org:/pub/freeradius/freeradius-server-3.0.13.tar.bz2\">Version 3.0.13</a> <a href=\"ftp://ftp.freeradius.org:/pub/freeradius/freeradius-server-3.0.13.tar.bz2.sig\">(sig)</a> , with incremental\nimprovements added in every release.</p>\n\n<hr>\n\n<h2>Documentation</h2>\n\n<p>Beginners to RADIUS should read\nthe <a href=\"http://networkradius.com/doc/FreeRADIUS%20Technical%20Guide.pdf\">FreeRADIUS\nTechnical Guide</a>.  Chapter 2 describes RADIUS concepts, such as\n\"authorization\", \"accounting\", and how the NAS and server communicate.\ni.e. how RADIUS works.  Chapter 3 describes how to install FreeRADIUS.\nChapter 4 describes what configuration files are distributed with the\nserver, and what they are used for.  This includes describing the\npacket processing lists (<code>request</code>, <code>reply</code>,\netc.), and the various configuration sections\n(<code>authorize</code>, <code>authenticate</code>, etc.)</p>\n\n<p>We recommend reading Chapter 4 to anyone who wants to understand\nhow the server works.</p>\n\n<p>There is extensive documentation for the server available via\nthe <a href=\"http://networkradius.com/doc/current/\">Network RADIUS\nDocumentation</a> page.  The \"introduction\" and \"concepts\" chapters\nare not available there yet, as those subjects are covered in the\nTechnical guide.</p>\n\n<p>We recommend in particular\nthe <a href=\"http://networkradius.com/doc/3.0.13/raddb/mods-available/\">modules</a>\npage, which contains documentation for nearly every module distributed\nwith the server.  We also recommend\nthe <a href=\"http://networkradius.com/doc/3.0.13/unlang/keywords.html\">Unlang\nkeywords</a> page, which describes what each keyword does, and how it\nis used.</p>\n\n<p>There is also documentation available on\nthe <a href=\"doc/\">documentation</a> page, and on\nthe <a href=\"http://wiki.freeradius.org/\">wiki</a>.</p>\n\n<hr>\n\n<h2>Stable Release</h2>\n\n<p>The \"stable release\" of FreeRADIUS\nis <a href=\"version3.html\">version 3</a>.  The goal of this version is\n<b>stability</b>.  Version 3 introduces a number of major changes over\nVersion 2.  The debugging output is clearer (and colourised!), more\nerrors are found while the configuration files are parsed instead of\nat run-time, RADIUS over TLS (i.e. RadSec) and RADIUS over TCP are\nsupported.  The <tt>raddb/</tt> directory has been re-arranged so that\nfiles are easier to find.</p>\n\n<p>Download: <a href=\"ftp://ftp.freeradius.org:/pub/freeradius/freeradius-server-3.0.13.tar.bz2\">3.0.13</a></p>\n\n<p>Upgrading to v3 from v2?
