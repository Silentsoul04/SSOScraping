font-family: arial, sans-serif;</pre>\n<pre>&nbsp; border-collapse: collapse;</pre>\n<pre>&nbsp; width: 100%;</pre>\n<pre>}</pre>\n<pre>td,th {</pre>\n<pre>&nbsp; border: 1px solid #dddddd;</pre>\n<pre>&nbsp; text-align: left;</pre>\n<pre>&nbsp; padding: 8px;</pre>\n<pre>}</pre>\n<pre>tr:nth-child(even) {</pre>\n<pre>&nbsp; background-color: #dddddd;</pre>\n<pre>}</pre>\n<p>Now, the HTML table will look like this,</p>\n<p><img class=\"wp-image-2342 aligncenter\" alt=\"Screen Shot 2017-03-21 at 10.15.52 AM\" src=\"http://www.learningjquery.com/wp-content/uploads/Screen-Shot-2017-03-21-at-10.15.52-AM-1024x585.png\" width=\"614\" height=\"351\"></p>\n<p>Let’s add the CSS classes required for highlighting the clicked column, adding up or down symbols, indicating the sorting order and a CSS class to highlight columns on mouse over event:</p>\n<pre>.focus {</pre>\n<pre>&nbsp; background-color: #ff00ff;</pre>\n<pre>&nbsp; color: #fff;</pre>\n<pre>&nbsp; cursor: pointer;</pre>\n<pre>&nbsp; font-weight: bold;</pre>\n<pre>}</pre>\n<pre>.selected {</pre>\n<pre>&nbsp; background-color: #ff00ff;</pre>\n<pre>&nbsp; color: #fff;</pre>\n<pre>&nbsp; font-weight: bold;</pre>\n<pre>}</pre>\n<pre>.asc:after {&nbsp; content: \"\\25B2\"; }</pre>\n<pre>.desc:after { content: \"\\25BC\"; }</pre>\n<h3><b>jQuery Code</b></h3>\n<p>To sort the HTML table, the solution would be to get the complete table data in an array and then perform the sort operation on the array based on the clicked column and considering the sort order.</p>\n<p>Below is the complete jQuery code:</p>\n<pre>$(document).ready(function() {</pre>\n<pre>&nbsp; $('th').each(function(col) {</pre>\n<pre>&nbsp; &nbsp; $(this).hover(</pre>\n<pre>&nbsp; &nbsp; function() { $(this).addClass('focus'); },</pre>\n<pre>&nbsp; &nbsp; function() { $(this).removeClass('focus'); }</pre>\n<pre>&nbsp; );</pre>\n<pre>&nbsp; &nbsp; $(this).click(function() {</pre>\n<pre>&nbsp; &nbsp; &nbsp; if ($(this).is('.asc')) {</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; $(this).removeClass('asc');</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; $(this).addClass('desc selected');</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; sortOrder = -1;</pre>\n<pre>&nbsp; &nbsp; &nbsp; }</pre>\n<pre>&nbsp; &nbsp; &nbsp; else {</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; $(this).addClass('asc selected');</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; $(this).removeClass('desc');</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; sortOrder = 1;</pre>\n<pre>&nbsp; &nbsp; &nbsp; }</pre>\n<pre>&nbsp; &nbsp; &nbsp; $(this).siblings().removeClass('asc selected');</pre>\n<pre>&nbsp; &nbsp; &nbsp; $(this).siblings().removeClass('desc selected');</pre>\n<pre>&nbsp; &nbsp; &nbsp; var arrData = $('table').find('tbody &gt;tr:has(td)').get();</pre>\n<pre>&nbsp; &nbsp; &nbsp; arrData.sort(function(a, b) {</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; var val1 = $(a).children('td').eq(col).text().toUpperCase();</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; var val2 = $(b).children('td').eq(col).text().toUpperCase();</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; if($.isNumeric(val1) &amp;&amp; $.isNumeric(val2))</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; return sortOrder == 1 ? val1-val2 : val2-val1;</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; else</pre>\n<pre>&nbsp;&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; return (val1 &lt; val2) ? -sortOrder : (val1 &gt; val2) ? sortOrder : 0;</pre>\n<pre>&nbsp; &nbsp; &nbsp; });</pre>\n<pre>&nbsp; &nbsp; &nbsp; $.each(arrData, function(index, row) {</pre>\n<pre>&nbsp; &nbsp; &nbsp; &nbsp; $('tbody').append(row);</pre>\n<pre>&nbsp; &nbsp; &nbsp; });</pre>\n<pre>&nbsp; &nbsp; });</pre>\n<pre>&nbsp; });</pre>\n<pre>});</pre>\n<p>The following is the outline of the jQuery solution.</p>\n<p>First, check each table heading to see if it is hovered over. If any of the table headings are hovered over, then add .focus CSS class to it to highlight it. When the mouse is moved away from the heading column, remove the focus CSS class, which brings the column heading back to the initial style. This is done by attaching the hover event to column headings, to toggle the .focus CSS class.</p>\n<p>Since the column heading has to be clickable, attach a click event to each column heading. In the click event, first check for the current sort order. If the column heading is clicked for the first time, the table must be sorted in ascending order by that column, and if the column is clicked again, the table must be sorted in descending order by that column. We defined 2 CSS classes: asc and desc. These classes are responsible for adding up/down arrows to the column headings and also tell us about the current sorting order.</p>\n<p>Inside the click event, check if the column heading has asc class applied to it. If yes, then remove it and assign desc class. Along with this, apply the selected CSS class, which highlights the column on the UI. We also set the value -1 to the sortOrder variable. This sortOrder variable will be used while sorting in descending order. If the current sort order value is descending, then add asc CSS class and remove desc class. Also, set sortOrder value to 1.</p>\n<p>Once we’ve sorted the column, remove the asc and desc classes from all the column headings except the selected one. Create an array variable arrData to store the complete table data. Here, exclude the header row and get data only for TD elements using jQuery <a href=\"https://api.jquery.com/get/#get2\" class=\"broken_link\">get()</a>. The <a href=\"https://api.jquery.com/get/#get2\" class=\"broken_link\">get()</a>returns an array of all of the elements.</p>\n<p>Now, on the array variable arrData, call the sort function to sort items inside the array. The default implementation of the sort() method sorts the values as strings in alphabetical and ascending order. Since we want to sort on descending order also, the default implementation will not work.&nbsp; The default implementation also fails to accurately sort the numerical values. The sort function takes an optional parameter: a compare function. We can leverage this parameter to fix all of the above problems.&nbsp; If we define a comparison function, then a pair of values from the array will be repeatedly sent to the function until all the elements of the array are processed. In the comparison function we then write a statement considering the pair of values passed to it so that the function returns any of the following three values: &lt;0, =0, or &gt;0.</p>\n<ul>\n<li>When the function returns the value &lt;0, the second value (of the pair of array values sent to the function) is larger than the first value and hence must be pushed down the sorting order.</li>\n<li>When the function returns the value &gt;0, the first value is larger than the second value, so it must be pushed down the sorting order.</li>\n<li>When the function returns the value =0, it means there is no need to change the sorting order since the two values are same.</li>\n</ul>\n<p>When this sort function is complete, the arrData array will have all the rows sorted by the selected column and value assigned to the sortOrder variable for ascending/descending order. Finally, the sorted rows from the arrData array are retrieved and appended to the tbody element of the table for display.</p>\n<p>If you want to play around with the code yourself, check out the demo&nbsp;<a href=\"https://jsfiddle.net/bmfz1pau/embedded/result/\" target=\"_blank\" class=\"broken_link\">example</a> on jsFiddle!</p>\n<h3><b>Conclusion</b></h3>\n<p>To sum it up, this post provides an easy jQuery solution that enables sorting on the HTML table columns in ascending and descending order. The solution also takes care of issues with the default implementation of sort() function on an array. The best thing about this is, it is achieved without using any jQuery plugins (which are an overhead).</p>\n\t\t\t\t</div>\n\n\t\t\t\t<p class=\"postmetadata\">\n\t\t\t\tLevels: <a href=\"http://www.learningjquery.com/category/levels/intermediate\">Intermediate</a> |  \t\t\t\t<a href=\"http://www.learningjquery.com/2017/03/how-to-sort-html-table-using-jquery-code#comments\">No comments</a>\n\t\t\t\t</p>\n\t\t\t</div>\n\t  </div>\n\t\t\t\t<div class=\"page-nav after\">\n\t\t\t<div class=\"prev\"><a href=\"http://www.learningjquery.com/page/2\" class=\"replace\">Older Posts</a></div>\n\t\t\t<div class=\"next\"></div>\n\t\t</div>\n\n\t\n\t</div>\n\n<div id=\"sidebar\" style=\"visibility: visible;
